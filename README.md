##  Key Features

###  Farmer Dashboard
A simple dashboard showing sales, earnings, and pending orders.

###  Produce Listing
Farmers can upload produce photos, set quantities, and prices for buyers to view.

###  Logistics Management
Drivers can view available deliveries, accept tasks, and track deliveries in real-time.

###  Secure Payment
An escrow system ensures safe, transparent, and fast transactions between all parties.

### üó£ Voice-Assisted Guidance
Voice support and trusted agents help farmers who aren‚Äôt familiar with smartphones register and use the app easily.

##  The User Flow
1. User downloads the app.  
2. Farmer, buyer, or driver signs up and is onboarded.  
3. Farmers list produce ‚Üí Buyers order ‚Üí Drivers deliver.  
4. Payments are held in escrow until both sides confirm satisfaction.  
5. Ratings help build trust for future transactions.
